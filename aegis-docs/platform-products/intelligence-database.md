# Intelligence Database

The Malicious Wallet Intelligence Database is a cornerstone of the Aegis security infrastructure. It is a continuously updated registry of flagged wallets, known scammers, and highly risky contract addresses.

### Continuous Threat Cataloging
The database is built to detect and catalog threats before they can inflict widespread damage on users. It is not a static list; it grows dynamically. Every scan executed through the Aegis platform contributes valuable signal back into the database, creating a decentralized network of threat detection.

### Security Audits and Verification
Aegis actively cross-references and enriches raw on-chain data with industry-standard registries, including the Jupiter verified token list and Metaplex metadata. This integration ensures that users are provided with the most accurate and up-to-date verification statuses for tokens and wallets. 

By maintaining this intelligence database, Aegis ensures that a threat detected once becomes a threat flagged for all users across the platform.
